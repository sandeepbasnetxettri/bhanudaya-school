Testing Instructions

To test the student registration form with email notifications:

  • Click the "Test Registration Form" button below to open the enhanced registration form
  • Fill in all required fields in the form
  • Click "Register" to submit the form
  • In demo mode, you'll see a success message indicating that a notification would be sent to sandeepbasnetxettri@gmail.com
  • For actual email functionality, you'll need to:
    • Install PHPMailer using Composer
    • Configure Gmail SMTP settings in login_system/php/config/email.php
    • Set up a proper web server with PHP support

Email Configuration Notes

To enable actual email sending functionality:

  1. Install PHPMailer by running: composer require phpmailer/phpmailer
  2. Update your Gmail credentials in login_system/php/config/email.php:
  3. Generate an App Password for Gmail at Google App Passwords
  4. Replace the placeholder values with your actual credentials
  5. Ensure your web server supports PHP and has the mail function enabled

Upon successful registration, emails will be sent to:

  • Student: Confirmation email to their registered email
  • Administrator: Notification email to sandeepbasnetxettri@gmail.com
